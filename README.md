# Data Structures & Algorithms Revision Website
![](https://img.shields.io/github/stars/tristanbudd/dsa-revision-website.svg) ![](https://img.shields.io/github/forks/tristanbudd/dsa-revision-website.svg) ![](https://img.shields.io/github/issues/tristanbudd/dsa-revision-website.svg)

## Project Description
Educational platform focused on learning and revising Data Structures & Algorithms. It provides clear explanations, structured topics, and interactive examples to reinforce understanding.

The platform is suitable for students and programming enthusiasts, built with modern web technologies to ensure a smooth, responsive, and effective learning experience across devices.

> [!WARNING]
> I’m new to working with React, Vite, and Tailwind CSS. This project isn’t perfect, but it served as a valuable learning experience.

---

## Features
- Mobile-responsive design.
- Interactive code examples with syntax highlighting (Mostly from my dsa-practicals repository).
- Comprehensive topic coverage (Trees, Graphs, Sorting, etc.)
- Clean, learning-focused interface.
- SEO-friendly structure.
- Fast React-based architecture.
- Detailed breakdown of all the DSA topics covered in your project.

## Installation / Setup

1. **Clone the repository**:
    ```
    git clone https://github.com/tristanbudd/starwars-fan-website.git
    ```
    ```
    cd dsa-revision-website
    ```

2. **Install Dependencies** using NPM:
    ```
    npm install
    ```

3. **Run the dev environment** using Vite:
    ```
    npm run dev
    ```

4. **Open the page** using Web Browser: (Port may vary if taken)
    ```
    http://localhost:5173
    ```
